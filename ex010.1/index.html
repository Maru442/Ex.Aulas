Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá ultilizar em cada caso.

  • Parágrafos
    • Use a tag <p>
  • Títulos
    • Use a tag <h1>
  • Imagens
    • Use a tag <img>
  • Abreviações
    • Use a tag <abbr>
  • Código-fonte
    • Digite ! e dê ENTER
  • Listas númeradas
    • Para a criação dessas listas, use a tag <ol>
  • Lista com demarcadores
    • Para a criação dessas listas, use a tag <ul>
